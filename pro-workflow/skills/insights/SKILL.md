# Session Insights

Surface patterns from learnings and session history.

## Trigger

Use when asking "show stats", "how am I doing", "analytics", or "correction rate".

## What It Shows

- **Session Summary** - Duration, edits, corrections, learnings captured, context usage
- **Learning Analytics** - Top categories, most applied rules, recent and stale learnings
- **Correction Patterns** - Most corrected areas, trends, suggestions for new rules
- **Correction Heatmap** - By category, by project, adaptive quality gate thresholds
- **Productivity Metrics** - Avg session length, edits/session, correction rate trend

## Output

Formatted analytics report with actionable suggestions for which patterns to capture as permanent rules.
